#### Остановить Uvicorn

```
fuser -k 8000/tcp
```

#### Переход в директорию
```
cd /home/servicebot/service-bot
source venv/bin/activate
```

#### Скачать с Github
```
git pull
```
#### Перезапуск приложения для отладки

```
fuser -k 8000/tcp
cd /home/servicebot/service-bot
source venv/bin/activate
git pull
uvicorn app.main:app --host 127.0.0.1 --port 8000
```

#### Запуск приложения для прода
```
cd /home/servicebot/service-bot
source venv/bin/activate
nohup uvicorn app.main:app --host 0.0.0.0 --port 8000 > uvicorn.log 2>&1 &
```



```
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

```
uvicorn app.main:app --host 0.0.0.0 --port 8000
```
#### Прочитать содержимое таблицы master
```
sudo -u postgres psql -d servicebot -c "SELECT 'master', COUNT(*) FROM master UNION ALL SELECT 'service_request', COUNT(*) FROM service_request;"
```

```
sudo -u postgres psql -d servicebot -c "SELECT id, master_id, name, lastname, phone, is_active, is_admin, max_user_id FROM master WHERE max_user_id = 40398020;"
```
#### Дропнуть таблицу master

```
sudo -u postgres psql -d servicebot -c "TRUNCATE TABLE master CASCADE; ALTER SEQUENCE master_id_seq RESTART WITH 1;"
```
